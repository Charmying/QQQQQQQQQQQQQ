| Big O | 名稱 | 舉例程式 | 說明 |
|-|-|-|-|
| `O(1)` | 常數時間 | `return arr[0];` | 無論輸入多大，執行時間固定不變 |
| `O(log n)` | 對數時間 | 二分搜尋 | 資料每次都減半 |
| `O(n)` | 線性時間 | for 迴圈 | 輸入多大，時間就花多長 (每個元素走一次) |
| `O(n log n)` | 線性對數時間 | 快速排序、合併排序 | 分治法排序演算法的典型複雜度 (高效排序演算法)，例如：Merge Sort、Quick Sort 的平均情況 |
| `O(n²)` | 平方時間 | 雙層巢狀迴圈 | 每個元素與所有其他元素互動 (巢狀迴圈或暴力法)，例如：氣泡排序、選擇排序 |
| `O(2ⁿ)` | 指數時間 | 遞迴費波那契 (未最佳化) | 每次呼叫自己兩次，成倍增長 (資料稍多就爆炸性成長)，例如：遞迴解法的費波那契數列 |
| `O(n!)` | 階乘時間 | 全排列 | 組合與排列問題 (非常低)，例如：排列組合的暴力解法，像是旅行推銷員問題 TSP |
